# csdpy
Template for Certified Scrum Developer course with Python

Use
- python3
- virtualenv
- pip

Installation
- git clone https://github.com/jgabardini/csdpy
- cd csdpy
- activate your virtualenv (with python3)
- pip install -r requirements.txt

Test
- behave
- nosetests

Run
- python rover.py
- browse to localhost:5000
